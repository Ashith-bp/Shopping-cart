A simple command-line shopping cart system built using Python and MySQL.

## Features
- View Products
- Add to Cart
- View Cart & Total
- MySQL Database Support

## Setup
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Import the database schema:
   ```bash
   mysql -u root -p < schema.sql
   ```
3. Run the app:
   ```bash
   python main.py
   ```
